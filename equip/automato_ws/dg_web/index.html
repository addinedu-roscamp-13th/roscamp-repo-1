인터페이스 맵 dg_01

DCS(본체) 중심 연동. 링크 색: 초록=양쪽 UP, 회색=DOWN. 메시지가 오갈 때 링크가 깜박이며 메시지명·방향(화살표)이 표시됩니다.

시나리오 테스트 (E0 · E1 · E2)

시퀀스 다이어그램의 3개 시나리오를 항목별로 실행·판정합니다. dcs · acs · ddago · ddagi · dg_ai 가 모두 UP이어야 합니다. 각 항목 실행 → PASS/FAIL 과 근거 로그가 표시됩니다.

DG AI Service 연결 (TCP)

현재 접속대상: -
· dg_ai 시뮬을 켜면 active=sim(시뮬 IP)로, 끄면 active=real(실서버 IP)로 자동 전환됩니다.
· DCS는 이 값을 읽어 다음 분석 요청에서 해당 엔드포인트로 자동 재접속합니다. 실서버 IP는 팀원 서버 주소로 저장해 두세요.

메시지 로그 (DCS 시점)

DCS가 주고받은 모든 메시지를 시각순 하나의 스크롤로 표시(1.5초 갱신, 위=과거/아래=최신). 실행 버튼을 누르면 초기화됩니다. 보관 건.
